# Scenarios: Library Management, Sorting, and Filtering

## Contents

1. **[L-01-02: Library Tabs](./L-01-02.feature)**: User can toggle between Watchlist and Watched statuses.
2. **[L-03: Custom List CRUD](./L-03.feature)**: User can create, rename, and delete custom collections.
3. **[L-04-05: List-Entry Association](./L-04-05.feature)**: User can add/remove entries to/from custom lists and view them.
4. **[L-06: Deletion Integrity](./L-06.feature)**: Deleting a list preserves the entries it contained.
5. **[LF-01: Filter by Genre](./LF-01.feature)**: User can filter library entries by one or more genres.
6. **[LF-02: Filter by Media Type](./LF-02.feature)**: User can filter library entries by media type (Movies/TV Shows).
7. **[LF-03: Filter by Rating Range](./LF-03.feature)**: User can filter library entries by their personal rating range.
8. **[LF-04: Filter by Watch Status](./LF-04.feature)**: User can filter the selected custom list by watch status.
9. **[LF-05: Filter by Custom List](./LF-05.feature)**: User can filter Watchlist and Watched tabs by custom list membership.
10. **[LF-06: Filter Badge](./LF-06.feature)**: User can see a badge showing the count of active filter categories.
11. **[LF-07: Clear Filters](./LF-07.feature)**: User can clear all active filters with a single action.
12. **[LS-01: Sort by Date Added](./LS-01.feature)**: User can sort entries by the date they were added to the library.
13. **[LS-02: Sort by Title](./LS-02.feature)**: User can sort entries alphabetically by title (A-Z, Z-A).
14. **[LS-03: Sort by Release Year](./LS-03.feature)**: User can sort entries by their release year.
15. **[LS-04: Sort by User Rating](./LS-04.feature)**: User can sort entries by their personal rating.
16. **[LS-05: Default Sort](./LS-05.feature)**: Library defaults to newest-first date sorting and localizes the selected label.
17. **[LS-06: Sort Persistence](./LS-06.feature)**: User's sort selection survives page reload.
18. **[LU-01: SortDropdown Component](./LU-01.feature)**: Library header shows the localized sort control and expected options.
19. **[LU-02: FilterBar Integration](./LU-02.feature)**: Library filter controls appear below the tabs and stay sticky.
20. **[LU-03: FilterBar Customization](./LU-03.feature)**: FilterBar adapts its available filters to the current screen.
21. **[LU-04: Empty State](./LU-04.feature)**: A populated tab or list shows a filtered empty state when filters remove all results.
